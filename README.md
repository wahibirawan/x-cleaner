# X Cleaner
A browser extension that helps you delete tweets, retweets, and likes in bulk. Clean your X history and reset your feed with simple controls.

## Features
- **Mass Delete Tweets**  
  Select a date range or keyword. X Cleaner deletes tweets one by one.

- **Mass Delete Retweets**  
  Scan your timeline and remove old retweets automatically.

- **Mass Unlike**  
  Clean your Likes page with automated unlike actions.

- **Smart Filters**  
  Filter by date, keyword, or content type.

- **Auto Scroll Engine**  
  The extension scrolls the page to collect all target items.

- **Adjustable Delay**  
  Set a delay between actions to keep activity natural.

- **Live Progress**  
  See real time logs, counts, and a progress bar.

## Modes
- **Quick Clean**  
  One tap to clear Tweets, Retweets, or Likes.

- **Advanced Clean**  
  Full control with filters and custom rules.

## How It Works
1. Install the extension.  
2. Open x.com and log in, the go to your X profile. Use tab posts to delete posts and retweet, use tab likes to delete like (unlike)
3. Open X Cleaner from your browser toolbar.  
4. Choose a mode.  
5. Start cleaning. The extension processes each item locally.

## Privacy Policy
X Cleaner does not collect, store, or send any personal data.  
All actions run locally in your browser.  
No servers. No analytics. No tracking.
